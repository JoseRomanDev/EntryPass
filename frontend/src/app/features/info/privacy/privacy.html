Política de Privacidad

Cómo gestionamos tus datos dentro del entorno académico de EntryPass.

Gestión de Datos

En EntryPass, la privacidad de los datos es una prioridad técnica. Dado que es un proyecto académico, el tratamiento de los datos sigue estas pautas:

  • Transparencia: Solo solicitamos los datos necesarios para el funcionamiento de la demo (email, nombre ficticio).
  • Seguridad: Las contraseñas se almacenan mediante algoritmos de hash seguros en el backend de Symfony.
  • Uso: Los datos nunca serán compartidos con terceros ni utilizados con fines comerciales.
  • Eliminación: La base de datos puede ser reiniciada en cualquier momento como parte del mantenimiento del proyecto.

Cookies y Almacenamiento

Utilizamos JSON Web Tokens (JWT) almacenados en el navegador para gestionar tu sesión de forma segura y eficiente, garantizando que solo tú puedas acceder a tus entradas compradas.

Nota sobre RGPD

Este proyecto cumple con los principios fundamentales de protección de datos, adaptados a un entorno de desarrollo y pruebas para un Trabajo de Fin de Grado.
