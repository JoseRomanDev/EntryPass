Sobre EntryPass

Descubre la visión y la tecnología detrás de nuestra plataforma de eventos.

Nuestra Misión

EntryPass es un proyecto personal desarrollado como Trabajo de Fin de Grado (TFG) para el ciclo de Desarrollo de Aplicaciones Web (DAW). Nuestra misión es proporcionar una plataforma robusta y moderna para la gestión de entradas a eventos, facilitando la conexión entre organizadores y asistentes.

Consiste en una solución integral que permite explorar eventos, comprar entradas de forma segura y recibir un código QR único para acceder al recinto, todo con una experiencia de usuario premium.

Stack Tecnológico

EntryPass está construido utilizando las últimas versiones de las tecnologías más potentes del ecosistema web:

Frontend

Angular 21 con gestión de estado mediante Signals para una reactividad óptima y fluida.

Backend

Symfony 7 y PHP 8.4 con tipado estricto, siguiendo los principios de la Arquitectura Hexagonal.

Base de Datos

PostgreSQL 16 para una persistencia de datos relacional robusta y segura.

Mensajería

RabbitMQ junto con Symfony Messenger para el procesamiento asíncrono de tareas.

Seguridad

Autenticación basada en JWT (LexikJWT) para una comunicación segura entre frontend y backend.

Infraestructura

Contenerización con Docker y servidor Nginx configurado como Reverse Proxy.

Excelencia Técnica

El objetivo académico de este proyecto es demostrar el dominio de una arquitectura moderna basada en los estándares más exigentes de la industria:

Arquitectura Hexagonal

Desacoplamiento total entre la lógica de negocio y las implementaciones tecnológicas en el backend.

Procesamiento Asíncrono

Gestión eficiente de tareas pesadas como generación de QRs y PDFs mediante colas de RabbitMQ.

Alta Fidelidad (UI/UX)

Interfaz moderna con estética Premium Dark, diseñada para ofrecer la mejor experiencia visual.

Infraestructura Robusta

Contenerización completa con Docker, garantizando un despliegue consistente y escalable.

Hitos Alcanzados

  • Despliegue de Producción optimizado con Nginx.
  • Flujo de Compra completo con simulación de pasarela de pagos.
  • Sistema de validación de entradas mediante Códigos QR.
  • Dashboard administrativo refinado para la gestión de eventos.
  • Suite de pruebas automatizadas con PHPUnit para garantizar la calidad del código.

Nota Académica

Este proyecto ha sido diseñado cumpliendo estrictamente con la rúbrica y los requisitos del ciclo DAW, enfocándose en la escalabilidad, el mantenimiento y la seguridad de la información.
